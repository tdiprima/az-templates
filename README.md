# Azure Templates

Azure service templates and utilities for Python.

I put this together because working with Azure's many services means constantly re-learning the same setup patterns. Rather than digging through docs every time, I wanted ready-to-run templates that get me to a working starting point fast — and serve as a quick reference when I can't remember the exact syntax or steps.

## Structure

- **ml/** - Machine Learning workspace connection utilities
- **queries/** - OpenAI and Azure OpenAI query scripts (GPT, RAG)
- **resources/** - Azure resource management (VMs, storage, networks, resource groups)
